## Overview The `Button` component provides interactive elements with multiple variants, sizes, and states. It supports icons, navigation, and comprehensive styling options for a complete button system. ## Features - **Multiple Variants**: Primary, secondary, outlined, and text buttons - **Size Options**: Small, medium, and large sizes - **Icon Support**: Can include Font Awesome icons with left/right positioning - **State Management**: Disabled and loading states - **Navigation**: Can act as a link with the `href` attribute - **Custom Styling**: Full support for the `sx:` attribute system - **Theme Integration**: Consistent with the design system ## Usage ```html ``` ## Examples ### Button Variants ### Button Sizes ### Buttons with Icons ### Icon Positioning ### Button States ### Navigation Buttons
