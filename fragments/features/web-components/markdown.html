## Overview The `Markdown` component provides rich markdown rendering with custom renderers and framework integration. It uses the `marked.js` library with custom renderers that convert markdown elements to framework components, ensuring consistent styling and functionality. The component supports custom icon syntax, automatic indentation handling, and HTML entity unescaping for optimal markdown processing. ## Features - **Rich Markdown Support**: Full CommonMark specification support - **Custom Renderers**: Framework component integration for tables, icons, and more - **Icon Syntax**: Special `icon:IconName;color` syntax for inline icons - **Table Styling**: Automatic table styling with zebra stripes and headers - **Code Highlighting**: Syntax highlighting for code blocks - **HTML Safety**: Automatic HTML entity handling and sanitization - **Theme Integration**: Consistent styling with framework theme ## Usage ```html # Heading This is **bold** and *italic* text. ``` ## Basic Markdown ### Headings ```html # Heading 1 ## Heading 2 ### Heading 3 #### Heading 4 ##### Heading 5 ###### Heading 6 ``` **Description**: All six levels of markdown headings with consistent styling. ### Text Formatting ```html This is **bold text** and this is *italic text*. You can also use __bold__ and _italic_ syntax. This is ***bold and italic*** text. ~~Strikethrough text~~ is also supported. ``` **Description**: Various text formatting options including bold, italic, and strikethrough. ### Lists ```html - Unordered list item 1 - Unordered list item 2 - Nested item 2.1 - Nested item 2.2 - Unordered list item 3 1. Ordered list item 1 2. Ordered list item 2 1. Nested ordered item 2.1 2. Nested ordered item 2.2 3. Ordered list item 3 ``` **Description**: Both unordered and ordered lists with nested items. ### Links and Images ```html [Link to Google](https://www.google.com) [Internal Link](/features) ![Alt text](https://via.placeholder.com/150) ``` **Description**: Links and images with proper styling and functionality. ## Tables ### Basic Table ```html | Name | Age | City | |------|-----|------| | John | 25 | New York | | Jane | 30 | London | | Bob | 35 | Paris | ``` **Description**: Basic table with automatic styling and zebra stripes. ### Table with Icons ```html | Status | Task | Priority | |--------|------|----------| | icon:CheckCircle;green500 | Complete project | High | | icon:Construction;orange500 | Fix bugs | Medium | | icon:Error;red500 | Review code | Low | ``` **Description**: Table with custom icon syntax in cells. ### Complex Table ```html | Feature | Status | Description | Notes | |---------|--------|-------------|-------| | User Auth | icon:CheckCircle;green500 | Complete | Ready for production | | Data Export | icon:Construction;orange500 | In Progress | 80% complete | | API Integration | icon:Error;red500 | Blocked | Waiting for external API | | UI Updates | icon:CheckCircle;green500 | Complete | All components updated | ``` **Description**: Complex table with multiple columns and status icons. ## Code Blocks ### Inline Code ```html Use `console.log()` to debug your JavaScript code.
