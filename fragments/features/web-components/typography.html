## Overview The `Typography` component provides a comprehensive text rendering system with predefined variants, responsive design, and advanced styling capabilities. It supports the full `sx:` styling system with specialized shorthand properties for spacing and layout. ## Usage ```html Main Heading Body text with custom styling ``` ## Live Examples Heading 1 Heading 2 Heading 3 Heading 4 Heading 5 Heading 6 Body 1 - Main body text for paragraphs and general content. Body 2 - Secondary body text for less prominent content. Subtitle 1 - Section subtitles and medium emphasis text. Subtitle 2 - Smaller subtitles and labels. Caption - Small text for captions and metadata. Overline - Uppercase text for labels and categories. Strong - Bold text for emphasis. Em - Italic text for emphasis. Blockquote - Quoted text with special styling. ## Attributes | Attribute | Type | Required | Default | Description | |-----------|------|----------|---------|-------------| | `variant` | string | No | `body1` | Typography variant (h1-h6, body1-body2, subtitle1-subtitle2, caption, overline, strong, em, blockquote) | ## How It Works ### Typography Variants 1. **Heading Variants**: h1-h6 with predefined font sizes and weights 2. **Body Variants**: body1-body2 for paragraph and general text 3. **Specialized Variants**: subtitle, caption, overline for specific use cases 4. **Emphasis Variants**: strong, em, blockquote for text emphasis ### Spacing Shorthand 1. **Padding**: p, pt, pb, pl, pr, px, py for padding control 2. **Margin**: m, mt, mb, ml, mr, mx, my for margin control 3. **Value Mapping**: Numeric values map to rem units (1 = 0.5rem) 4. **Axis Shorthand**: px/py for horizontal/vertical spacing ### Style Processing 1. **SX Processing**: Processes sx: attributes for styling 2. **CSS Generation**: Generates CSS from sx attributes 3. **Style Application**: Applies styles to the component element 4. **Responsive Support**: Handles responsive styling conditions 5. **Theme Integration**: Integrates with framework theme system ## Common Use Cases - **Headings**: Page titles, section headers, and content hierarchy - **Body Text**: Paragraphs, descriptions, and main content - **Labels**: Form labels, captions, and metadata - **Navigation**: Menu items, breadcrumbs, and navigation text - **Status Messages**: Success, error, and warning messages - **Data Display**: Tables, lists, and structured data presentation
