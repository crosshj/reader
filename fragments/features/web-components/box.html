## Overview The `Box` component is a flexible container component that provides layout and styling capabilities. It's based on the Material-UI Box concept and supports the `sx:` attribute system for inline styling. ## Features - **Flexible Layout**: Supports flexbox, grid, and block layouts - **SX Styling**: Full support for the `sx:` attribute system - **Responsive Design**: Built-in responsive behavior - **Theme Integration**: Works seamlessly with the design system ## Usage ```html ``` ## Examples ### Basic Container This is a basic Box container with border and padding. ### Flex Layout Item 1 Item 2 Item 3 ### Grid Layout Grid Item 1 Grid Item 2 Grid Item 3
