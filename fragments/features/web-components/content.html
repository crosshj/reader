## Overview The `Content` component provides a flexible container for main page content. It serves as a wrapper that can be styled with the `sx:` attribute system and is commonly used as the main content area in page layouts. ## Features - **Flexible Container**: Can hold any content and be styled as needed - **SX Styling**: Full support for the `sx:` attribute system - **Layout Integration**: Designed to work within page layouts - **Semantic Structure**: Provides proper content hierarchy ## Usage ```html Your content here ``` ## Examples ### Basic Content ```html Basic content container ``` ### Styled Content ```html Styled content with background and shadow ``` ### Content with Different Sizes ```html Small content Large content with more padding ``` ### Content with Flex Layout ```html Title This is content with a title and description. ``` ## Common Use Cases - **Page Content**: Main content area for pages and sections - **Card Containers**: Wrapping content in cards and panels - **Form Sections**: Grouping form elements and labels - **Article Content**: Containing articles, blog posts, and documentation
